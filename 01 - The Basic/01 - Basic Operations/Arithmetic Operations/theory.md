The ARITHMETIC OPERATORS are used to perform basic math operations.

 ARITHMETIC OPERATORS:
 [+  ] - Addition        
 [-  ] - Subtraction    
 [*  ] - Multiplication 
 [/  ] - Division        
 [%  ] - Modulus        
 [++ ] - Increment      
 [-- ] - Decrement    

 EXAMPLES FOR EACH ONE:
 x + y
 x - y
 x * y
 x / y
 x % y
 ++x
 --x
